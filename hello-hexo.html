如何搭建一个Hexo博客

发表于 2018-11-18 | 更新于 2018-11-23 | 分类于 Hexo | 评论数： | 阅读次数：
本文字数： 661 | 阅读时长 ≈ 1 分钟

本博客采用 Hexo 3.8.0 搭建，使用了 NexT 6 主题，并使用 Github Pages 部署。在搭建过程中基本按照如下步骤进行：

  1. 使用 Github 空间搭建 Hexo 博客 1–安装篇（基于 IntelliJ IDEA）(我选用了 NexT 主题，并没有选用文中的 yelee 主题，所以文中主题的配置部分没有参考，如果和我一样使用 Github Pages 空间，则文中的绑定域名部分也可以不看)
  2. Hexo 的 _config.yml 配置文件配合 Hexo中文文档
  3. NexT 的 _config.yml 配置文件配合 NexT官方网站

还是强烈建议通过从头到尾看一遍 Hexo 和 NexT 的配置文件并配合其各自官方文档来配置博客，虽然要花不少时间但能把所有配置都熟悉一下。不过由于官方文档可能更新不及时，有的内容比较旧，所以最好配合你所使用的版本的其他用户的配置心得来看，搜索引擎大把。

当然，如果你想很快搭建一个可用的博客，日后再慢慢配置的话，也可以参考步骤1和下面的步骤4。

  1. [实战干货]30分钟快速搭建hexo3.7.0 + next主题6.4教程(持续更新)

另外，像取消打赏文字旋转动画这种在配置文件中无法配置，需要修改其他文件的操作就需要自行搜索一下了，使用关键字”Hexo+你想实现的功能”基本都能搜索到。

最后，由于 Hexo 博客的博文需要使用 Markdown 语言编辑，再附一篇 Markdown 入门，www.markdown.cn，首页的知识点很全，多写就熟了。
